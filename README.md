# graphql-koa
A starter for graphql with koa and koa-router.

## Getting Started
```
git clone https://github.com/ishan123456789/graphql-koa.git graphql-koa
```
```
cd graphql-koa
```

```
npm install
```
To start run
```
npm run start
```
